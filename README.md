# Minesweeper
This project is a command line implementation of the classic Minesweeper game, developed in Python. The game allows a player to uncover cells on a grid while avoiding hidden mines. The objective is to reveal all non-mine cells without triggering a mine. This implementation focuses on the use of appropriate data structures and algorithms, including 2D arrays, recursion, and stacks.

## Features
- Three difficulty levels (Easy, Medium, Hard)
- Grid-based game board
- Automatic expansion of empty cells (flood-fill algorithm)
- Undo and redo functionality
- Win and loss detection
- Input validation for user moves
